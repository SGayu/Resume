MOVIE SESSION

Spare Parts (2015) is an inspiring true-story film about a group of high school students from an underfunded school who, despite facing numerous obstacles, compete in a national robotics competition. Led by their unconventional teacher, Fredi, they overcome challenges like limited resources, personal struggles and skepticism from others. The film showcases themes of determination, teamwork and the power of education highlighting how passion and innovation can triumph over adversity. It is a testament to the impact of mentorship and the importance of believing in one's potential, no matter the odds.

Learning HTML and CSS basics is an exciting introduction to web development. HTML provides the structure of web pages, while CSS styles them. Through hands-on practice, you’ll quickly grasp how to create and design simple web pages, gaining confidence in building and customizing your own websites.Starting with HTML, we learnt how to create elements like headings, paragraphs, links and images while CSS allows to control layout, colors, fonts and spacing. The process can initially feel challenging, but as we build simple pages, the concepts start to click, and gained confidence in creating structured and well-designed websites.
