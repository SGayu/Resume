TINKERCAD

1.Understand Circuit Design – Learn to build electronic circuits using a drag-and-drop interface with components like resistors, LEDs and transistors.

2.Perform Real-time Simulation – Test and analyze circuit behavior by observing voltage, current and component interactions in a simulated environment.

3.Integrate Arduino – Simulate Arduino boards, write and upload C++ code, and develop embedded projects without physical hardware.

4.Learn Block-based Programming – Utilize visual, drag-and-drop code blocks for beginner-friendly programming and prototyping.

5.Work with Breadboard Layouts – Design circuits using a breadboard view to mimic real-world prototyping techniques.

ARDUINO UNO

1.Understand Microcontroller Boards – Learn about microcontroller-based boards like the ATmega328P and their role in interactive electronics projects.

2.Explore Digital I/O Pins – Understand the functionality of 14 digital I/O pins, including PWM outputs for controlling various electronic components.

3.Learn About Analog Input Pins – Gain knowledge of 6 analog input pins and their use in reading sensor data (e.g., temperature, light).

4.Understand USB Connectivity – Explore how USB is used for programming the microcontroller and supplying power.

5.Power Supply Options – Learn about different power sources (USB or external 7-12V) for microcontroller operation.

6.Utilize the Built-in LED – Understand the use of the onboard LED on pin 13 for basic testing and debugging.
